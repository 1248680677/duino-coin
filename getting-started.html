Download the latest release

Before you start doing anything with Duino-Coin and you don't intend on using the Web Services you have to download the latest release containing everything you need.
To do so, visit latest releases page on our GitHub, scroll down through the changelog and download release a for your OS.

Downloads section in the releases page

After downloading, unzip the zip archive (for example to your desktop) and proceed with the next steps.
Notice: The binaries in the release section are precompiled for 64-bit systems (most new computers).
If you want to launch them on another CPU architecture however (e.g. 32-bit PC), you need to launch them from the source.

Register an account on Duino-Coin network

The first thing you have to do in order to use Duino-Coin is to register your account on the nework.

To do so, you can either use the GUI Wallet (recommended), CLI Wallet (both of them are included in every Duino-Coin release you've downloaded in the previous step) or Web Wallet to register though your browser.

Everything is made to be as simple as possible, so you should have no trouble with registering.
Just launch any of the wallets, select the register option and fill in the blanks.

Start mining with your computer

Mining with your computer is the easiest to setup way of earning Duino-Coin.

To do so, simply launch the PC_Miner (included in every Duino-Coin release you've downloaded in the first step).
If you're launching it for the first time, it will ask you some details like your Duino-Coin username, number of threads, etc. - you can set them how you like or stick to the recommended values.

Configuration tool when launching the miner for the first time

When you successfully finish creating the config, miner will automatically start mining.

You are getting rewarded for every Accepted and Block found you see. The share numbers and percentage show your miner effectiveness. The kH/s (or MH/s) values tell the speed of mining. Diff means the difficulty you're mining at and ping is just the time of network delays.

Start mining with your AVR (Arduino) board

Mining with AVR boards is the main idea of Duino-Coin - there is no other coin that supports it.

We assume you have Arduino IDE already installed on your PC, if you don't - do that right now. We need it to upload Duino-Coin code to your board.
In every Duino-Coin release you'll find a folder called Arduino_Code and in it, a file named Arduino_Code.ino - open it with Arduino IDE.
When Arduino IDE finishes loading, select your board (Tools > Board...), port (Tools > Port...) and upload the code.

Upload button

After uploading the code, launch AVR_Miner program (yet again, included in every Duino-Coin release).
The rest is similar to setting up the PC Miner. Config tool will ask you a few questions needed to organize everything and when it's finished, it will start mining automatically.

Configuration tool when launching the miner for the first time

Similarly to the PC Miner, you are getting rewarded for every Accepted and Block found you see. The share numbers and percentage show your avr board effectiveness. Next value in seconds tells how long finding the share took. The H/s values tell the speed of mining. Diff means the difficulty you're mining at and ping is just the time of network delays.

Start mining with your Raspberry Pi

Raspberry Pi boards are great for mining - they can run headlessly, don't use a lot of power and are very powerful.

We assume you are using Raspbian or another Debian-based distro on your Raspi, if not, then these instructions may not work.
First thing you have to do is to open the Terminal. In it, paste the following commands, one by one:
sudo apt install python3 python3-pip git # Install python3 and git
git clone https://github.com/revoxhere/duino-coin # Clone duino-coin repo
cd duino-coin
python3 PC_Miner.py # Launch the official miner
 And that's it! If everything was successful, the Miner should start with the configuration tool, after responding to a few questions it'll ask you your Raspberry Pi should start mining automatically.

Raspberry Pi... mining!


Start mining with your ESP8266/ESP32 (or compatible) board

Mining with ESP boards allows you to mine headlessly - you don't need to have them connected to a computer as they have their own Wi-Fi capabilities that connect to the server.

Similarly to the Arduino mining, you have to locate the folder named ESP8266_Code (or ESP32_Code) and open ESP8266_Code.ino (or ESP32_Code.ino) file with Arduino IDE configured to support ESP boards.
Before you click upload, you need to change a few things here. Near the top you should see lines which you need to edit according to your needs:
const char* ssid = "WiFi name"; // Change this to your WiFi SSID
const char* password = "WiFi password"; // Change this to your WiFi password
const char* ducouser = "DUCO username"; // Change this to your Duino-Coin username
After changing them, select your board (Tools > Board...), port (Tools > Port...) and upload the code to your board.

Upload button

After uploading the code it is a good idea to open the Arduino Serial Monitor (Tools > Serial Monitor), set its baud rate to 115200 and see how's the ESP doing. If everything is correct, you should see messages saying that a share was accepted.

ESP8266... mining!


Mining on other devices

You can mine on other devices that have internet access (for example Smartphones, Smart TVs and more).
To do that, you can simply open the Duino-Coin Web Miner on the device you want to mine on.
Enter your username, click Start mining and you're good to go!
