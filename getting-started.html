Getting started

Download latest release

Click this link to visit page containing releases with Wallet, PC Miner, Arduino Miner & Code.

(example release)

Locate version suited for your OS (probably Windows) and click on it to download.

After downloading, right click on the .zip file and click on an option that says something like "Extract".

Register using wallet

To start doing anything with Duino-Coins, you need to firstly register on network.

Launch Wallet and proceed to register proceed to register by following program tips.

After successfull registration you will be able to login. You will also receive 0.5 DUCO as a bonus for signing up!

Mine using PC

To mine with your computer, launch PC Miner , enter your account credentials and set CPU usage in %.

If you enter everything correctly, miner will remember the settings and automatically start mining Duino-Coins when you launch it.


You can also use PoT Miner  to mine passively and get rewarded after some amount of time.

Steps in launching it are the same as in PC Miner.

Mine using Arduino

To mine using Arduino (or compatible) board, locate Arduino_code folder and Arduino_code.ino file in it.

Open this file with Arduino IDE and upload it to your board just like LED blinker codes you were uploading in the past. No need to change anything here.

After you've uploaded the code, launch Arduino_Miner and enter your account credentials and Arduino COM port (the same your board is on, e.g. COM8).

If you've entered everything correctly, good job! Your Arduino will now start making some money - mining Duino-Coins.

Mine using Android phone

Download Minimal-PC_Miner onto your phone.

Then open it using an app called PyDroid3 that you can download from Google Play.

You will need to change username and password lines near the top. When you're done, just launch it.
