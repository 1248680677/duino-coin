About Duino-Coin

There are many cryptocurrencies available. Most of them can be mined, but only using powerful CPUs, GPUs or ASICs to make some real money.
I think this goes against statement that currency should be for everyone.
I've also wanted to do something related to cryptocurrencies with Arduino because nobody has done that yet.
That's why Duino-Coin was created.
First ever cryptocurrency that can be profitably mined with regular home computers, smartphones, Arduino boards and everything that can hash SHA.

 

Duino-Coin is:

centralized - no need to download gigabytes of blockchains and you can be sure that you won't lose your funds!

• available worldwide - all you need is internet access

• cost effective - why? read >link to blog post here<

• open-source - anyone can contribute - Duino-Coin GitHub

 

Why should you mine Duino-Coin?

• DUCO (Duino-Coin) is worth ~0.1 XMG (Coin Magi) at the time I'm writing this. You can exchange it here.

• Mining Duino-Coins isn't very CPU intensive.

• It can be mined using various platforms.

• Mining on regular computers, smartphones etc. is profitable.

 

Why would you want to mine on Arduino?

• You may have some Arduino boards that don't do anything.

• Watching Arduino do something really useful that produces money is satisfying.

• Because you can.
